Important: This app is distributed directly (not via app stores yet). Follow the steps below for your device. Android users need to allow unknown sources; iPhone users need a special build method.

Android

APK direct install · Android 6.0+

  • 1

    On your Android phone, go to Settings → Security (or Apps → Special App Access) and enable Install unknown apps for your browser.

  • 2

    Tap the button below to download the .apk file. If your browser warns you, tap Download anyway.

  • 3

    Open your Downloads folder, tap the APK file, then tap Install. The app will appear on your home screen.

⬇ Download APK (Android)
No APK file yet? The uploaded project is Flutter source code — you need to build the APK first using flutter build apk --release, then host the output file and link it here.
🍎

iPhone / iPad

iOS 12+ · Requires special distribution

  • 1

    Option A — TestFlight: If the developer has set up TestFlight, tap the invite link they shared with you to install the beta app directly.

  • 2

    Option B — Ad-hoc / Enterprise: Download the .ipa file below. You'll need to trust the developer profile under Settings → General → VPN & Device Management.

  • 3

    Option C — App Store: Once the app is published, search "AgriApp" in the App Store and install it normally.

🔗 Join TestFlight (iOS) ⬇ Download IPA (Ad-hoc)
Note for iOS: Apple does not allow direct IPA installs without a paid developer account ($99/yr) and a signed build. The easiest option for sharing with testers is TestFlight.
